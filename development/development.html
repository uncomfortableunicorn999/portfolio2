The Project

In this project, I learned how to use javascript and state. The task was to create a page that had cards on it. Given that it was right before Thnaksgiving, and I love pie, I decided to make a pie website. The features include filtering and sorting and adding a recipe to a favorites list.

New Concepts and Features

  • I used state to figure out which checkbox was selected
  • After knowing which boxes were selected, I filtered the list using python's filter method
  • Intuitive Filtering
    • You can also compound filters, so if two author filters are selected, it shows results for both of those authors
    • But if two filters of different categories are selected, it shows the intersection of those categories
    • When no filters are selected, all results are shown
    • This was the most intuitive way to filter

  • I had two sorting features, alphabetical and by amount of time
  • I learned how to create sorting functions in python, and use the sort method for lists
  • I used state to see which radio button was selected, so it would sort appropriately.

  • There is a bookmark feature that allows user to add a specific recipe to their favorites list
  • The bookmark can be checked and unchecked
  • A list of favorited items along with the total time to cook all of the favorited items is displayed, the list updates automatically depending on what is checked
